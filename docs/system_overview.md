# Dojo Manager System Overview

## Purpose
Preserve martial arts techniques using AI + biomechanics analysis.

## Key Components
- **Pose Analysis**: 3D pose extraction from videos
- **Biomechanics Engine**: Physical metrics calculation
- **ML Models**: Technique classification and feedback
- **LLM Integration**: Natural language coaching
- **Adaptive Configuration**: Environment-aware profiles

## Unique Features
- Multi-GPU support (ROCm/CUDA/Vulkan)
- Hardware-optimized training parameters
- Real-time feedback with <100ms latency
- Professional error handling and security

## Use Cases
- Technique preservation for aging masters
- Remote coaching platforms
- Performance analysis for athletes
- Historical martial arts documentation

## Technical Foundation
- Python 3.11+ with async/GPU acceleration
- Nix flakes for reproducible environments
- Gradio UI for visualization
- Type-safe configuration management
